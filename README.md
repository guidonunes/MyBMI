# MyBMI 📊

A sleek, modern Android application designed to calculate and track Body Mass Index (BMI). Built to help users monitor their body composition and conditioning metrics over time.

* **UI Framework:** Jetpack Compose

## ✨ Features

* **Modern UI:** Built entirely with Jetpack Compose for a smooth, reactive user experience.
* **Data Persistence:** Saves calculation history using Firebase Database so users can track their progress over time.

## 🚀 Getting Started
To run this project locally on your machine:

1. Clone this repository: `git clone https://github.com/guidonunes/MyBMI.git`
2. Open the project in Android Studio.
3. Connect the app to your Firebase project and drop your `google-services.json` file into the `app/` directory.
4. Sync Gradle and run the app on an emulator or physical device.
